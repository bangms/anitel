# ANITEL (반려동물 호텔 예약 사이트)
  - 반려동물 호텔 정보를 이용자에게 제공하고, 필요할 시 예약하는 기능을 제공하는 사이트 (프로젝트 간략한 소개)
  
## 팀원 (각자 블로그나 깃이나 뭐주소? 맡은 부분 소개)
  - 최홍재 (조장) : 관리자 마이페이지 
  - 김하영 : 일반회원 마이페이지 전체, 결제 페이지
  - 노현호 : 사업자 마이페이지 전체
  - 방윤지 : 메인, 검색리스트, 사업자 회원가입, 로그인페이지, 예약페이지
  - 이다희 : 게시판 전체, 호텔 상세페이지, 일반 회원가입
  - 이영인 : 관리자 마이페이지
